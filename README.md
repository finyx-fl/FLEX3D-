# FLEX3D Engine 🌊

A lightweight 3D engine built with Three.js + Jolt Physics.  
Made for easy web 3D with ocean, sky, and physics out of the box.

## Installation

Import it directly from CDN:

```html
<script type="module">
  import * as FLEX from 'https://cdn.jsdelivr.net/gh/finyx-fl/FLEX3D-@main/index.js'
</script>
